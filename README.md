# 🛡️ ISO 27001 Audit Toolkit

Este repositorio contiene un conjunto de herramientas, plantillas y listas de verificación diseñadas para facilitar la ejecución de auditorías internas de Seguridad de la Información bajo el marco de la norma **ISO/IEC 27001**.

## 🎯 Objetivo
Proporcionar recursos estandarizados que ayuden a los profesionales de ciberseguridad y auditores a:
- Evaluar el cumplimiento de los controles del Anexo A.
- Documentar hallazgos (No Conformidades, Observaciones, Oportunidades de Mejora) de manera estructurada.
- Gestionar y presentar métricas de riesgo técnico y operativo.

## 📂 Estructura del Repositorio

- **/01-Checklists:** Listas de verificación técnica para evaluar controles específicos (ej. Control de accesos, Seguridad física, Operaciones).
- **/02-Matrices-Riesgo:** Formatos y ejemplos de evaluación y tratamiento de riesgos de seguridad de la información.
- **/03-Reportes-y-Hallazgos:** Plantillas metodológicas para la correcta redacción y presentación de informes de auditoría.

## 🚀 Cómo usar este toolkit
Siéntete libre de clonar o descargar este repositorio para adaptar los formatos a las necesidades específicas de tu organización o proyecto de consultoría.
